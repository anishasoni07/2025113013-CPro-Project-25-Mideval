# 2025113013-CPro-Project-25

This repository contains my code submission for the CPro Project Mideval.

# Files:
**twixtmideval.c** - Contains the complete code for implementation of the game interface.

# How to compile:
Open the terminal and run the following command -
1. gcc twixtmideval.c

# How to run:
Open the terminal and run the following command -
1. ./a.out

# How to play:
1. Enter normal coordinates in the form of 'x y' when asked on each player's respective turn. Please use the given indices to correctly make your move.
2. To resign, enter -1 -1
3. To start a new game, enter 0 0
4. Main grid indices run from 1 to 22.
5. Edge row/column indices are 0 and 23.
